## 前置作業
* [Google Chrome](https://www.google.com/chrome/) **必須** 安裝於系統中
## 理論上支援的作業系統
* Windows 7 或以上
* Windows 11 (Windows on ARM)
* macOS High Sierra (10.13) 或以上
* Linux
## 通過測試的作業系統
* Windows 7
* Windows 8
* Windows 8.1
* Windows 10
* Windows 11
* Windows 11 (Windows on ARM)
* macOS Ventura (Apple Silicon)
* Ubuntu 22.10
